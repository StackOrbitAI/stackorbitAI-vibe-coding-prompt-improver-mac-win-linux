## ⚡ How It Works

1. **Type Anywhere**: Type your raw coding prompt in any application (VS Code, Cursor, ChatGPT, Claude.ai, terminal, text editor, etc.).
2. **Press Global Shortcut**: Press `Ctrl+Shift+Space` (or `Cmd+Shift+Space` on Mac).
3. **AI Enhancement Overlay**: Floating HUD pops up near your cursor, captures selected text, and optimizes it with role definition, context, and output formatting.
4. **Auto-Paste**: The improved prompt is automatically pasted right back into your active input field.
