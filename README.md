## ⚡ How It Works

1. **Type Anywhere**: Type your raw coding prompt in any app (VS Code, Cursor, ChatGPT, Claude.ai, terminal, etc.).
2. **Press Shortcut**: Press `Ctrl+Shift+Space` (or `Cmd+Shift+Space` on Mac).
3. **AI Enhancement**: App overlay HUD pops up, captures your text, and optimizes it with structured context and role definition.
4. **Auto-Paste**: The improved prompt is automatically pasted right back into your active window.
