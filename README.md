## ✨ Features & Major Enhancements in v1.0.15

1. **🌟 Prompt Library & Default Mode Preset Selector**:
   - Added **Prompt Library** tab in Settings.
   - Choose your default active mode used whenever global hotkey (`Ctrl+Shift+P` / `Ctrl+Shift+X`) is pressed:
     - **Improve & Structure Prompt (Vibe Coding Standard)**
     - **Enhanced Prompt & Technical Precision**
     - **Improve Existing Prompt (Refine & Polish)**
     - **Always Correct Prompt Without Meaning Change**
     - **Update & Modernize Prompt (Latest Standards)**
     - **Correct Prompt Grammar in English**
     - **Strict Meaning Preservation (Minimal Fixes)**
2. **Automated Background Update Service**: Periodically checks for new releases and handles background update downloads.
3. **Taskbar Management**: Automatically cleans up tray icon and removes app from taskbar during updates for a seamless transition.
4. **Release Notes & Feature Screenshots Showcase**: Dedicated section in Settings UI to view latest release notes and feature previews.
5. **Previous Release Direct Download Links & Shortcuts**: Download links for previous releases directly accessible inside the UI.
