## ✨ Features & Fixes in v1.0.17

1. **Blank Screen Crash Fix**: Resolved browser bundle externalized module dependency issue by separating frontend types into pure `src/types/settings.ts`. Settings UI now loads instantly and renders 100% cleanly.
2. **Custom Prompt Preset Builder**: Create, save, and manage your own custom system instructions inside the Prompt Library tab (e.g., *"I want to correct my prompt in English without meaning change"*).
3. **Active Preset Selector**: Select any built-in or custom library prompt as default; whenever global hotkeys (`Ctrl+Shift+P` / `Ctrl+Shift+X`) are pressed, the AI dynamically applies your active default instruction!
